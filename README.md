# after-ai-text

An OpenClaw skill for rewriting AI-assisted drafts into cleaner human text.

## What it does

- removes neural clichés and fake-depth phrasing
- catches structural AI patterns and cringe rhetoric
- helps rewrite text into a more natural human voice
- supports multiple content lengths and categories
- includes user-calibrated anti-pattern references

## Structure

- `SKILL.md` — core instructions and routing
- `references/` — rules, anti-patterns, examples, format matrix
- `assets/` — prompt and output templates

## Use cases

- humanize AI-assisted drafts
- audit text for neural smell
- rewrite posts, intros, reflections, and expert content
- reduce template-like phrasing in Russian text

Built for iterative real-world calibration.
