# Style Sampling

Use this file when the user provides prior writing samples.

## Minimum useful sample set

Prefer 5-10 texts by the same author in the same genre.

## Extract

- average sentence length and rhythm variation
- typical openings and turns
- sharpness vs softness of judgments
- amount of colloquial language
- preferred level of detail
- recurring rhetorical habits
- what the author never says

## Important distinction

Do not mimic surface quirks only.

Match:
- pace
- pressure
- level of directness
- density of specifics
- tolerance for roughness

Not just:
- favorite words
- filler particles
- punctuation habits in isolation

## Safety rule

Never claim an exact voice clone from weak samples. If samples are thin, say the rewrite is only loosely style-informed.

## User-specific calibration

If the user actively rejects certain phrases or sentence shapes during iteration, record them in a dedicated anti-pattern file and treat those judgments as higher priority than generic web advice.
